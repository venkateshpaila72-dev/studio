# **App Name**: Shadow Strike

## Core Features:

- Shuriken Toss: Tap to launch shurikens at enemies. Timing and accuracy are crucial for defeating foes.
- Endless Dash: Automatically dash through procedurally generated levels, testing reflexes and agility.
- Obstacle Avoidance: Dodge and navigate past increasingly challenging obstacles to survive as long as possible.
- High Score Tracking: Record the player's best score, encouraging replayability and competition.
- Adaptive Difficulty: Adjust the game's difficulty based on the player's skill, increasing enemy spawn rates and obstacle complexity over time to push player limits.

## Style Guidelines:

- Primary color: Deep red (#8B0000) to evoke danger and action, fitting for a ninja theme.
- Background color: Dark charcoal (#1A1A1A) for a stealthy, shadowy ambiance.
- Accent color: Silver (#C0C0C0) for highlighting key elements like shurikens and scores, providing contrast and visual interest.
- Font: 'Space Grotesk' for a techy, action-oriented feel in headlines; 'Inter' for a neutral feel in the body
- Use sharp, stylized icons for enemies, power-ups, and score displays.
- A dynamic and responsive layout that adjusts to various screen sizes for mobile and desktop play.
- Fast-paced and impactful animations for shuriken throws, enemy defeats, and obstacle dodges, enhancing the arcade feel.
